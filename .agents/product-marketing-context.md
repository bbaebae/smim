# Product Marketing Context

*Last updated: 2026-04-29*

## Product Overview
**One-liner:** 읽은 것을 진짜로 기억하게 해주는 AI 지식 관리 서비스
**What it does:** URL 하나를 붙여넣으면 AI가 핵심 내용을 자동 요약하고, SM-2 간격 반복 알고리즘으로 최적 타이밍에 복습 알림을 보내 장기 기억을 돕는다. 주간 이메일 리포트로 지식 성장을 눈으로 확인할 수 있다.
**Product category:** Second Brain / 지식 관리 (PKM) 도구
**Product type:** SaaS (B2C)
**Business model:** 프리미엄 구독 — Free(₩0, 월 20개 저장), Pro(₩6,900/월), Annual(₩59,000/년)

## Target Audience
**Target users:** 정보 소비량은 많지만 기억 보유율이 낮아 고민하는 지식 근로자·학습자
**Decision-makers:** 개인 (셀프 구매)
**Primary use case:** 아티클·유튜브·PDF 등 읽은 콘텐츠를 저장하고, AI 요약 + 간격 반복으로 실제 기억에 남기기
**Jobs to be done:**
- "나중에 봐야지"하고 잊어버린 콘텐츠를 체계적으로 관리하고 싶다
- 읽은 내용이 기억에 남지 않아서 시간 낭비라는 느낌을 없애고 싶다
- 흩어진 지식을 한 곳에 모아 필요할 때 꺼내 쓰고 싶다
**Use cases:**
- 대학원생·직장인이 논문·아티클 요약 및 복습
- 유튜브 강의를 저장해 핵심만 빠르게 파악
- PDF 교재·보고서를 업로드해 요약 생성

## Problems & Pain Points
**Core problem:** 정보가 넘쳐나는 시대에 읽고 나면 금방 잊는다. "나중에 봐야지"로 미루다 아예 잊어버리거나, 읽었는데 기억이 안 나서 다시 찾는 일이 반복된다.
**Why alternatives fall short:**
- 노션·Obsidian: 저장은 되지만 복습 기능이 없어 결국 안 읽는다
- 북마크 앱(Pocket 등): 요약·AI 분석 없이 링크만 쌓인다
- 플래시카드(Anki): 카드 직접 만들어야 해서 진입 장벽이 높다
**What it costs them:** 읽는 시간 투자 대비 기억 보유율 낮음 → 지식 성장 없음, 같은 내용 반복 검색, 시간 낭비감
**Emotional tension:** "이걸 읽는 게 의미 있나?" 하는 회의감, 바쁜 일상에서 자기계발을 못 하고 있다는 불안감

## Competitive Landscape
**Direct:** Readwise — 하이라이트 기반, 영어 중심, 가격 비쌈 (월 $7.99~)
**Secondary:** Notion AI, Obsidian — 수동 정리 필요, 복습 알고리즘 없음
**Indirect:** 북마크 앱(Pocket, Instapaper) — 저장만 되고 AI 요약·복습 없음 / Anki — 카드 직접 제작 부담

## Differentiation
**Key differentiators:**
- URL 하나로 AI 요약 + SM-2 복습이 자동으로 세팅됨 (설정 불필요)
- 한국어 특화 (인터페이스·AI 요약·이메일 모두 한국어)
- 아티클·유튜브·PDF·텍스트 5가지 콘텐츠 유형 통합 지원
- 주간 이메일 리포트로 "내가 얼마나 성장했나" 가시화
**How we do it differently:** 저장과 동시에 AI 요약 + 복습 스케줄이 자동 생성. 사용자는 카드 만들거나 태그 달 필요 없음
**Why that's better:** 마찰 제로 → 실제로 꾸준히 쓰게 됨
**Why customers choose us:** 한국어 환경에서 Readwise 대체재 없음, 설정 없이 바로 시작 가능, 합리적 가격

## Objections
| Objection | Response |
|-----------|----------|
| "또 안 쓰게 될 것 같아서" | 알림이 알아서 오기 때문에 사용자가 기억하지 않아도 된다. 수동 입력 없음 |
| "이미 노션에 정리하고 있는데" | 노션은 저장 도구, 스밈은 기억 도구. 병행 가능하며 노션 링크도 저장 가능 |
| "AI 요약 품질이 어떤가요?" | Claude(Anthropic) 기반으로 고품질 한국어 요약 제공, 무료로 직접 체험 가능 |

**Anti-persona:** 이미 Anki를 잘 쓰는 사람, 영어 콘텐츠만 소비하는 사람, 저장보다 검색 중심 워크플로우를 선호하는 사람

## Switching Dynamics
**Push:** 노션/북마크에 콘텐츠가 쌓이는데 다시 안 읽게 됨, 읽은 내용이 기억에 남지 않는 허탈함
**Pull:** URL 붙여넣기만 하면 AI가 다 해준다는 편의성, 복습 알림이 알아서 온다는 자동화
**Habit:** "어차피 정리 앱은 쓰다 말았다"는 경험, 기존 북마크 앱·노션 루틴
**Anxiety:** "또 한 달 쓰다 말 것 같다", "AI 요약이 부정확하면 어떡하지", "개인 정보 보안"
